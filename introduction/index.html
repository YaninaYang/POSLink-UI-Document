    Introduction

    Welcome to the POSLink UI Design and User Guide! This guide will tell you what is POSLink UI and teach you how to build your Android apps using the POSLink UI library.

    The purpose of design POSLink UI is to let BroadPOS applications to offload the user interfaces used by end users of POS terminals to third-party UI Applications. The UI App collects necessary data from the end users and submit them to BroadPOS. However, the UI App does not get access to sensitive data, and they are handled by BroadPOS App.

    POSLink UI is an SDK which includes all the constants needed for an UI App to maintain communication with BroadPOS. The constants include Activity and Broadcast actions, name of keys to contain values, and list of valid values for categorical data. All the constants that are provided in the SDK are discussed along with when they are used.
