    Common Confirmation Intents #

    This page describes all implict common intents that BroadPOS app can use to perform actions to confirm action.

    Start UI #

    BroadPOS might show status like CardStatus.CARD_REMOVAL_REQUIRED before start any Activity Action.
    In such case, it’s hard to choose default ui or customized ui to show status. So we add this action.
    In insert any time case, if customized app implements this action, BroadPOS will call customized app to foreground.
    Suggest prompt message “Processing,Please wait…”
    After this activity/dialog onResume, send output to BroadPOS let it knows you are ready to show status.

    Action
    ConfirmationEntry.ACTION_START_UI

    Category
    ConfirmationEntry.CATEGORY

    Input Extras

    • EntryExtraData.PARAM_PACKAGE
      The package name of BroadPOS app which started activity
      Type: String
      Mandatory

    Ouput Extras

    • EntryRequest.PARAM_ACTION
      Action
      The value is ConfirmationEntry.ACTION_START_UI Type: String
      Mandatory

    Example Intent Filter

    <activity ...>
    <intent-filter>
        <action android:name="com.pax.us.pay.action.START_UI" />
        <category android:name="com.pax.us.pay.ui.category.CONFIRMATION" />
        <category android:name="android.intent.category.DEFAULT" />
    </intent-filter>
</activity>

    Example Send Output to BroadPOS

    Bundle bundle = new Bundle();
bundle.putString(EntryRequest.PARAM_ACTION, ConfirmationEntry.ACTION_START_UI);

Intent intent = new Intent(EntryRequest.ACTION_NEXT);
intent.putExtras(bundle);
intent.setPackage(packageName); //Package name of BroadPOS app which started activity
context.sendBroadcast(intent);

    Is Card Present? #

    Action
    ConfirmationEntry.ACTION_CHECK_CARD_PRESENT

    Category
    ConfirmationEntry.CATEGORY

    Input Extras

    • EntryExtraData.PARAM_PACKAGE
      The package name of BroadPOS app which started activity
      Type: String
      Mandatory

    • EntryExtraData.PARAM_TRANS_TYPE
      Transaction type. Example: “CREDIT SALE”
      Type: String
      Mandatory

    • EntryExtraData.PARAM_TRANS_MODE
      The transaction mode BroadPOS is running in.
      Default Value is TransMode.NORMAL.
      Type: String
      Mandatory
      The possible values are:

      • TransMode.NORMAL
      • TransMode.DEMO
      • TransMode.TEST
      • TransMode.TEST_AND_DEMO

    • EntryExtraData.PARAM_TIMEOUT
      Timeout (unit: ms).
      Default value is 30000.
      Type : Long
      Mandatory

    • EntryExtraData.PARAM_OPTIONS
      Options
      Positive option: ConfirmationType.YES
      Negative option: ConfirmationType.NO
      Type: String[]
      Mandatory

    Ouput Extras

    • EntryRequest.PARAM_CONFIRMED
      Confirmation result
      If select ConfirmationType.YES, return TRUE
      If select ConfirmationType.NO, return FALSE
      Type: Booelan
      Mandatory

    • EntryRequest.PARAM_ACTION
      Action
      The value is ConfirmationEntry.ACTION_CHECK_CARD_PRESENT Type: String
      Mandatory

    Example Intent Filter

    <activity ...>
    <intent-filter>
        <action android:name="com.pax.us.pay.action.CONFIRM_CARD_PRESENT" />
        <category android:name="com.pax.us.pay.ui.category.CONFIRMATION" />
        <category android:name="android.intent.category.DEFAULT" />
    </intent-filter>
</activity>

    Example Send Output to BroadPOS

    Bundle bundle = new Bundle();
bundle.putString(EntryRequest.PARAM_ACTION, ConfirmationEntry.ACTION_CHECK_CARD_PRESENT);
bundle.putBoolean(EntryRequest.PARAM_CONFIRMED, confirmation); //confirmation result

Intent intent = new Intent(EntryRequest.ACTION_NEXT);
intent.putExtras(bundle);
intent.setPackage(packageName); //Package name of BroadPOS app which started activity
context.sendBroadcast(intent);

    Exmaple Convert QRCode Conent

     private Bitmap createQRCode(public static final String content, int qrCodeSize, Bitmap logo){
    Matrix m = new Matrix();
    int imageWidth = 80;
    int imageHeight = 80;
    m.setScale((float) imageWidth/logo.getWidth(), (float)imageHeight/logo.getHeight());
    logo = Bitmap.createBitmap(logo, 0, 0,
            logo.getWidth(), logo.getHeight(), m, false);
    MultiFormatWriter writer = new MultiFormatWriter();
    Hashtable<EncodeHintType, Object> hst = new Hashtable<>();
    hst.put(EncodeHintType.CHARACTER_SET, "UTF-8");
    BitMatrix matrix = null;
    try {
        matrix = writer.encode(content, BarcodeFormat.QR_CODE, qrCodeSize, qrCodeSize, hst);
    } catch (Exception e) {
        Logger.e(e);
        return null;
    }
    int width = matrix.getWidth();
    int height = matrix.getHeight();
    int halfW = width / 2;
    int halfH = height / 2;
    int[] pixels = new int[width * height];

    for (int y = 0; y < height; y++) {
        for (int x = 0; x < width; x++) {
            if (x > halfW - imageWidth/2 && x < halfW + imageWidth/2
                    && y > halfH - imageHeight/2 && y < halfH + imageHeight/2) {
                pixels[y * width + x] = logo.getPixel(x - halfW
                        + imageWidth/2, y - halfH + imageHeight/2);
            } else {
                if (matrix.get(x, y)) {
                    pixels[y * width + x] = 0xff000000;
                }
            }

        }
    }
    logo.recycle();
    Bitmap bitmap = Bitmap.createBitmap(width, height, Bitmap.Config.ARGB_8888);
    bitmap.setPixels(pixels, 0, width, 0, 0, width, height);
    return bitmap;
}
