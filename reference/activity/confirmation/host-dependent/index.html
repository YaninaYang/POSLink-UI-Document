    Host Dependent Confirmation Intents #

    This page describes all host-dependent implict intents that BroadPOS app can use to perform actions to confirm action.

    Card Will Be Deactivated, Are You Sure?? #

    Action
    ConfirmationEntry.ACTION_CHECK_DEACTIVATE_WARN

    Category
    ConfirmationEntry.CATEGORY

    Input Extras

    • EntryExtraData.PARAM_PACKAGE
      The package name of BroadPOS app which started activity
      Type: String
      Mandatory

    • EntryExtraData.PARAM_TRANS_TYPE
      Transaction type. Example: “CREDIT SALE”
      Type: String
      Mandatory

    • EntryExtraData.PARAM_TRANS_MODE
      The transaction mode BroadPOS is running in.
      Default Value is TransMode.NORMAL.
      Type: String
      Mandatory
      The possible values are:

      • TransMode.NORMAL
      • TransMode.DEMO
      • TransMode.TEST
      • TransMode.TEST_AND_DEMO

    • EntryExtraData.PARAM_TIMEOUT
      Timeout (unit: ms).
      Default value is 30000.
      Type : Long
      Mandatory

    • EntryExtraData.PARAM_OPTIONS
      Options
      Positive option: ConfirmationType.YES
      Negative option: ConfirmationType.NO
      Type: String[]
      Mandatory

    Ouput Extras

    • EntryRequest.PARAM_CONFIRMED
      Confirmation result
      If select ConfirmationType.YES, return TRUE
      If select ConfirmationType.NO, return FALSE
      Type: Booelan
      Mandatory

    • EntryRequest.PARAM_ACTION
      Action
      The value is ConfirmationEntry.ACTION_CHECK_DEACTIVATE_WARN Type: String
      Mandatory

    Example Intent Filter

    <activity ...>
    <intent-filter>
        <action android:name="com.pax.us.pay.action.CONFIRM_DEACTIVATE_WARN" />
        <category android:name="com.pax.us.pay.ui.category.CONFIRMATION" />
        <category android:name="android.intent.category.DEFAULT" />
    </intent-filter>
</activity>

    Example Send Output to BroadPOS

    Bundle bundle = new Bundle();
bundle.putString(EntryRequest.PARAM_ACTION, ConfirmationEntry.ACTION_CHECK_DEACTIVATE_WARN);
bundle.putBoolean(EntryRequest.PARAM_CONFIRMED, confirmation); //confirmation result

Intent intent = new Intent(EntryRequest.ACTION_NEXT);
intent.putExtras(bundle);
intent.setPackage(packageName); //Package name of BroadPOS app which started activity
context.sendBroadcast(intent);

    Confirm Surcharge Fee #

    Action
    ConfirmationEntry.ACTION_CONFIRM_SERVICE_FEE

    Category
    ConfirmationEntry.CATEGORY

    Input Extras

    • EntryExtraData.PARAM_PACKAGE
      The package name of BroadPOS app which started activity
      Type: String
      Mandatory

    • EntryExtraData.PARAM_TRANS_TYPE
      Transaction type. Example: “CREDIT SALE”
      Type: String
      Mandatory

    • EntryExtraData.PARAM_TRANS_MODE
      The transaction mode BroadPOS is running in.
      Default Value is TransMode.NORMAL.
      Type: String
      Mandatory
      The possible values are:

      • TransMode.NORMAL
      • TransMode.DEMO
      • TransMode.TEST
      • TransMode.TEST_AND_DEMO

    • EntryExtraData.PARAM_TIMEOUT
      Timeout (unit: ms).
      Default value is 30000.
      Type : Long
      Mandatory

    • EntryExtraData.PARAM_CURRENCY
      Currency type. Default value is CurrencyType.USD
      It is used to display a amount value.
      Example:
      The value of amount is 100, and currency type is CurrencyType.USD, the amount on page is shown as “$1.00”
      else, if currency type is CurrencyType.POINT, the amount on page is shown as “POINT 100”.
      Type: String
      Mandatory
      The possible values are:

      • CurrencyType.USD
      • CurrencyType.POINT
      • CurrencyType.CAD
      • CurrencyType.EUR
      • CurrencyType.GBP

    • EntryExtraData.PARAM_TOTAL_AMOUNT
      Total Amount
      Type: Long
      Mandatory

    • EntryExtraData.PARAM_SERVICE_FEE
      Service Fee Type: Long
      Mandatory

    • EntryExtraData.PARAM_OPTIONS
      Options
      Positive option: ConfirmationType.YES
      Negative option: ConfirmationType.NO
      Type: String[]
      Mandatory

    Ouput Extras

    • EntryRequest.PARAM_CONFIRMED
      Confirmation result
      If select ConfirmationType.YES, return TRUE
      If select ConfirmationType.NO, return FALSE
      Type: Booelan
      Mandatory

    • EntryRequest.PARAM_ACTION
      Action
      The value is ConfirmationEntry.ACTION_CONFIRM_SERVICE_FEE
      Type: String
      Mandatory
