    POSLink Manage UI Intents #

    This page describes all implict intents that BroadPOS app can use to perform Manage UI request from POSLink.

    #

    Action
    OptionEntry.$1

    Category
    OptionEntry.CATEGORY

    Input Extras

    • EntryExtraData.PARAM_PACKAGE
      The package name of BroadPOS app which started activity
      Type: String
      Mandatory

    • EntryExtraData.PARAM_TRANS_TYPE
      Transaction type. Example: “CREDIT SALE”
      Type: String
      Mandatory

    • EntryExtraData.PARAM_TRANS_MODE
      The transaction mode BroadPOS is running in.
      Default Value is TransMode.NORMAL.
      Type: String
      Mandatory
      The possible values are:

      • TransMode.NORMAL
      • TransMode.DEMO
      • TransMode.TEST
      • TransMode.TEST_AND_DEMO

    • EntryExtraData.PARAM_TIMEOUT
      Timeout (unit: ms).
      Default value is 30000.
      Type : Long
      Mandatory

    • EntryExtraData.PARAM_OPTIONS
      Options
      Possibile option:

      Type : String[]
      Mandatory

    Ouput Extras

    • EntryRequest.PARAM_INDEX
      Selected Index
      Value start from 0
      Type: Integer
      Mandatory

    • EntryRequest.PARAM_ACTION
      Action
      The value is OptionEntry.$1 Type: String
      Mandatory

    Example Intent Filter

    <activity ...>
    <intent-filter>
        <action android:name=$2 />
        <category android:name="com.pax.us.pay.ui.category.OPTION" />
        <category android:name="android.intent.category.DEFAULT" />
    </intent-filter>
</activity>

    Example Send Output to BroadPOS

    Bundle bundle = new Bundle();
bundle.putString(EntryRequest.PARAM_ACTION, OptionEntry.$1);
bundle.putInt(EntryRequest.PARAM_INDEX, index); //Selected index

Intent intent = new Intent(EntryRequest.ACTION_NEXT);
intent.putExtras(bundle);
intent.setPackage(packageName); //Package name of BroadPOS app which started activity
context.sendBroadcast(intent);
