    Common Text Intents #

    This page describes all common implict intents that BroadPOS app can use to perform actions to enter text.

    Enter Amount #

    Action
    TextEntry.ACTION_ENTER_AMOUNT

    Category
    TextEntry.CATEGORY

    Input Extras

    • EntryExtraData.PARAM_PACKAGE
      The package name of BroadPOS app which started activity
      Type: String
      Mandatory

    • EntryExtraData.PARAM_TRANS_TYPE
      Transaction type. Example: “CREDIT SALE”
      Type: String
      Mandatory

    • EntryExtraData.PARAM_TRANS_MODE
      The transaction mode BroadPOS is running in.
      Default Value is TransMode.NORMAL.
      Type: String
      Mandatory
      The possible values are:

      • TransMode.NORMAL
      • TransMode.DEMO
      • TransMode.TEST
      • TransMode.TEST_AND_DEMO

    • EntryExtraData.PARAM_TIMEOUT
      Timeout (unit: ms).
      Default value is 30000.
      Type : Long
      Mandatory

    • EntryExtraData.PARAM_CURRENCY
      Currency type. Default value is CurrencyType.USD
      It is used to display a amount value.
      Example:
      The value of amount is 100, and currency type is CurrencyType.USD, the amount on page is shown as “$1.00”
      else, if currency type is CurrencyType.POINT, the amount on page is shown as “POINT 100”.
      Type: String
      Mandatory
      The possible values are:

      • CurrencyType.USD
      • CurrencyType.POINT
      • CurrencyType.CAD
      • CurrencyType.EUR
      • CurrencyType.GBP

    • EntryExtraData.PARAM_VALUE_PATTERN
      Length limit for input value.
      Default value is “1-12”. It means min length is 1, max length is 12.
      Type: String
      Mandatory

    Ouput Extras

    • EntryRequest.PARAM_AMOUNT
      Amount
      Type: Long
      Mandatory

    • EntryRequest.PARAM_ACTION
      Action
      The value is TextEntry.ACTION_ENTER_AMOUNT
      Type: String
      Mandatory

    Example Intent Filter

    <activity ...>
    <intent-filter>
        <action android:name="com.pax.us.pay.action.ENTER_AMOUNT" />
        <category android:name="com.pax.us.pay.ui.category.TEXT" />
        <category android:name="android.intent.category.DEFAULT" />
    </intent-filter>
</activity>

    Example Send Output to BroadPOS

    Bundle bundle = new Bundle();
bundle.putString(EntryRequest.PARAM_ACTION, TextEntry.ACTION_ENTER_AMOUNT);
bundle.putLong(EntryRequest.PARAM_AMOUNT,value); //AMOUNT entered

Intent intent = new Intent(EntryRequest.ACTION_NEXT);
intent.putExtras(bundle);
intent.setPackage(packageName); //Package name of BroadPOS app which started activity
context.sendBroadcast(intent);

    Enter Tip #

    Action
    TextEntry.ACTION_ENTER_TIP

    Category
    TextEntry.CATEGORY

    Input Extras

    • EntryExtraData.PARAM_PACKAGE
      The package name of BroadPOS app which started activity
      Type: String
      Mandatory

    • EntryExtraData.PARAM_TRANS_TYPE
      Transaction type. Example: “CREDIT SALE”
      Type: String
      Mandatory

    • EntryExtraData.PARAM_TRANS_MODE
      The transaction mode BroadPOS is running in.
      Default Value is TransMode.NORMAL.
      Type: String
      Mandatory
      The possible values are:

      • TransMode.NORMAL
      • TransMode.DEMO
      • TransMode.TEST
      • TransMode.TEST_AND_DEMO

    • EntryExtraData.PARAM_TIMEOUT
      Timeout (unit: ms).
      Default value is 30000.
      Type : Long
      Mandatory

    • EntryExtraData.PARAM_CURRENCY
      Currency type. Default value is CurrencyType.USD
      It is used to display a amount value.
      Example:
      The value of amount is 100, and currency type is CurrencyType.USD, the amount on page is shown as “$1.00”
      else, if currency type is CurrencyType.POINT, the amount on page is shown as “POINT 100”.
      Type: String
      Mandatory
      The possible values are:

      • CurrencyType.USD
      • CurrencyType.POINT
      • CurrencyType.CAD
      • CurrencyType.EUR
      • CurrencyType.GBP

    • EntryExtraData.PARAM_BASE_AMOUNT
      Base Amount
      Type: Long
      Mandatory

    • EntryExtraData.PARAM_TIP_NAME
      Tip Name
      In BroadPOS standalone mode, if multiple tips are enabled, value might be “Tip”, “Tip2”, “Tip3” or any value configured by merchant.
      Default value is “Tip”.
      Type: String
      Mandatory

    • EntryExtraData.PARAM_TIP_UNIT
      Tip Unit
      Default value is UnitType.CENT
      Type: String
      Mandatory
      The possible values are:

      • UnitType.CENT
      • UnitType.DOLLAR

    • EntryExtraData.PARAM_VALUE_PATTERN
      Length limit for input value.
      Default value is “0-12”. It means min length is 0, max length is 12.
      Type: String
      Mandatory

    • EntryExtraData.PARAM_TIP_OPTIONS
      Tip Options
      Used when tip select is enabled.
      Example:
      if value of EntryExtraData.PARAM_TIP_OPTIONS is {“300”,“500”,“700”},
      you could provide 3 options for user:

      Select Tip
      $3.00
      $5.00
      $7.00

      Type: String[]
      Optional

    • EntryExtraData.PARAM_TIP_RATE_OPTIONS
      Tip Rate Options
      Used when tip select is enabled.
      Used with EntryExtraData.PARAM_TIP_OPTIONS together.
      Example:
      if value of EntryExtraData.PARAM_TIP_OPTIONS is {“1000”,“1500”,“1800”} and
      value of EntryExtraData.PARAM_TIP_RATE_OPTIONS is {“10”,“15”,“18”},
      you could provide 3 options for user:

      Select Tip
      $10.00(10%)
      $15.00(15%)
      $18.00(18%)

      Type: String[]
      Optional

    • EntryExtraData.PARAM_TIP_NAMES
      Name of Enabled Tips
      Used when multiple tips are enabled.
      Type: String[]
      Optional

    • EntryExtraData.PARAM_TIP_AMOUNTS
      Value of Enabled Tips
      Used when multiple tips are enabled.
      Used with EntryExtraData.PARAM_TIP_OPTIONS together.
      Example:
      If TIP1, TIP2, TIP3 are all enabled. Base amount is 10000.
      If name of enabled tips is: {“TIP”, “TIP2”, “TIP3}
      value of enabled tips is: {”", “200”,""}
      That means: The customer has bypass TIP, entered $2.00 TIP2. TIP3 not valued yet.

      Amount $10.00
      TIP $0.00
      TIP2 $2.00
      TIP3 $0.00

      Else, if value of enabled tips is: {“0”, “200”,""}
      That means: The customer has enterer $0.00 TIP, entered $2.00 TIP2. TIP3 not valued yet.

      Amount $10.00
      TIP $0.00
      TIP2 $2.00
      TIP3 $0.00

      Type: String[]
      Optional

    • EntryExtraData.PARAM_ENABLE_NO_TIP_SELECTION
      TRUE means NO TIP SELECTION is enabled.
      Type: Boolean
      Optional

    Ouput Extras

    • EntryRequest.PARAM_TIP
      Tip
      If return nothing, BroadPOS will treat it as tip bypassed.
      If return 0, BroadPOS will treat it as “NO TIP”
      Type: Long
      Optional

    • EntryRequest.PARAM_ACTION
      Action
      The value is TextEntry.ACTION_ENTER_TIP
      Type: String
      Mandatory

    Example Intent Filter

    <activity ...>
    <intent-filter>
        <action android:name="com.pax.us.pay.action.ENTER_TIP" />
        <category android:name="com.pax.us.pay.ui.category.TEXT" />
        <category android:name="android.intent.category.DEFAULT" />
    </intent-filter>
</activity>

    Example Send Output to BroadPOS

    Bundle bundle = new Bundle();
bundle.putString(EntryRequest.PARAM_ACTION, TextEntry.ACTION_ENTER_TIP);
bundle.putLong(EntryRequest.PARAM_TIP,value); //TIP entered

Intent intent = new Intent(EntryRequest.ACTION_NEXT);
intent.putExtras(bundle);
intent.setPackage(packageName); //Package name of BroadPOS app
context.sendBroadcast(intent);

    Example Send Output to BroadPOS (Bypass TIP)

    Bundle bundle = new Bundle();
bundle.putString(EntryRequest.PARAM_ACTION, TextEntry.ACTION_ENTER_TIP);
